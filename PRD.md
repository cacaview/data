## 一、项目概述

### 1.1 项目名称
东盟跨境贸易AI智能分析平台
(ASEAN Cross-Border Trade AI Analytics Platform, ACTAP)

### 1.2 一句话定位
基于多源公共数据与AI大模型的中国-东盟跨境贸易智能分析与决策平台

### 1.3 项目价值主张
通过融合海关总署、UN Comtrade、ASEANstats等9个权威公开数据源，结合AI趋势预测、商品聚类分析和RCEP关税计算引擎，为中国-东盟跨境贸易企业提供从市场洞察、选品决策、关税优化到风险预警的一站式智能服务。

## 二、项目背景

### 2.1 政策背景

2025-2026年是中国-东盟数字经济合作的制度性突破期：

- 中国-东盟自贸区3.0版：2024年底完成实质性谈判，2026年数字经济规则全面实施
- 东盟数字经济框架协议(DEFA)：2025年10月吉隆坡宣布实质性完成谈判，2026年底正式签署，预计至2030年东盟数字经济规模将达2万亿美元
- RCEP生效三周年：中国-东盟跨境电商交易额年均增速超20%
- 科技部《面向东盟人工智能赋能发展科技能力提升行动三年工作方案(2025-2027)》：明确合作开发本地语言大模型、建设AI联合实验室
- 广西《推动工业高质量数据集建设实施方案》：到2028年打造100个工业高质量数据集

### 2.2 行业痛点

中国-东盟跨境贸易面临的核心挑战：

- 多语言本地化：需覆盖11种东盟语言，营销/客服/合规成本高
- 物流效率低：东盟地区基础设施参差不齐，岛屿国家配送难
- 跨境支付割裂：各国支付系统不统一，本地化支付方式碎片化
- 关税计算复杂：RCEP原产地规则复杂，各国法规差异大
- 选品决策难：信息不对称，主要靠经验，缺乏数据支撑
- 市场风险高：政策变化、汇率波动、贸易摩擦等缺乏预警

### 2.3 竞品分析

- 通用BI工具(Tableau/PowerBI)：缺乏贸易领域知识，数据需手动整理
- 单一数据源平台(如海关数据网)：数据维度单一，缺乏AI分析
- 大厂跨境电商平台(阿里国际站、Lazada)：主要服务大客户，AI能力未对外开放
- 高校同类项目：多集中于智慧政务/城市治理，跨境贸易AI分析方向空白

### 4.1 总体架构

### 4.2 模块A：多源贸易数据融合引擎

#### A1. 数据采集
- 数据源接入：海关总署API、UN Comtrade API、ASEANstats、广西统计年鉴、商务部东盟数据、RCEP关税库、北部湾港数据、广西公共数据平台、ITC Trade Map
- 采集方式：API实时调用 + 定时爬虫 + 文件上传
- 数据格式：CSV、JSON、Excel、API JSON
- 采集频率：核心数据每日更新，辅助数据每周更新

#### A2. 数据清洗
- 字段标准化（HS编码统一、商品名称映射、国家代码统一）
- 缺失值处理（插值/删除/标记）
- 异常值检测（基于3σ原则和Isolation Forest）
- 数据去重与一致性校验

#### A3. 数据融合
- 实体对齐（同一商品/国家的多源匹配）
- 时序对齐（不同时间粒度数据对齐）
- 多维建模（贸易-商品-国家-时间四维数据立方体）
- 增量更新机制

### 4.3 模块B：AI智能分析引擎

#### B1. 贸易趋势预测
- 算法：LSTM时序预测模型 + Prophet季节性分解
- 预测对象：未来12个月中国-东盟双边贸易额、商品结构
- 输出：趋势曲线、置信区间、关键转折点识别
- 精度要求：MAPE < 15%

#### B2. 商品结构聚类
- 算法：K-Means + DBSCAN + 层次聚类
- 分析维度：贸易额、增长率、市场份额、价格弹性
- 输出：商品群组、相似商品推荐、潜力爆品识别
- 价值：辅助选品决策

#### B3. RCEP关税智能计算
- 算法：基于规则引擎 + 决策树
- 输入：原产地、商品HS编码、目标国、价值
- 输出：最优关税税率、累积规则适用性、优惠原产地判定
- 价值：相比手工计算，准确率提升至98%+，效率提升90%

#### B4. 贸易风险预警
- 算法：时序异常检测 (Isolation Forest) + 规则引擎
- 监测对象：贸易额异动、政策变化、汇率波动、关税调整
- 输出：风险等级、预警原因、应对建议
- 价值：实现"事前预警、事中处置、事后分析"

#### B5. 市场空白点识别
- 算法：关联规则挖掘 + 协同过滤
- 分析维度：东盟10国×HS编码二级类目的贸易密度
- 输出：蓝海市场TOP榜单、增长潜力指数
- 价值：发现"一带一路"新机遇

### 4.4 模块C：交互式数据可视化大屏

#### C1. 中国-东盟贸易全景地图
- 基于Mapbox的中国-东盟10国贸易热力地图
- 支持按贸易额、商品类别、时间范围筛选
- 鼠标悬停显示国家详情面板
- 双向数据流箭头动画

#### C2. 10国贸易对比看板
- 多维雷达图：贸易额、增速、依存度、商品多样性
- 国别排行条形图
- RCEP利用率对比

#### C3. 商品结构分析
- 桑基图：商品类别流向变化
- 词云图：东盟国家热搜商品
- 折线图：商品贸易额时序变化

#### C4. AI预测曲线展示
- 多模型预测对比（实际值 vs LSTM vs Prophet）
- 预测区间可视化
- 关键事件标注

#### C5. 物流路线图
- 中国-东盟主要港口连接
- 北部湾港吞吐量时序图
- 集装箱流向路径

### 4.5 模块D：AI智能决策助手

#### D1. 对话式分析
- 基于大语言模型的自然语言查询
- 支持"越南2024年从中国进口最多的3C产品是什么"等业务问题
- 自动生成数据查询SQL/Pandas代码
- 结果以可视化图表+文字解读呈现

#### D2. 智能报告生成
- 一键生成贸易分析报告（PDF/Word）
- 自动提炼关键洞察
- 报告模板：市场分析报告、选品建议报告、风险评估报告

#### D3. 个性化推荐
- 基于用户历史行为推荐分析场景
- 订阅贸易动态（新品类、新政策、新机会）
- 邮件/微信推送

### 4.6 模块E：数据资产管理

#### E1. 数据血缘追踪

#### E2. 数据质量监控

### 5.2 数据模型

核心数据表设计：

```
fact_trade (贸易事实表)
- trade_id, year, month, reporter, partner, hs_code
- trade_value_usd, trade_value_rmb, quantity, unit
- source, created_at, updated_at

dim_country (国家维度)
- country_code, country_name_cn, country_name_en
- asean_member, rcep_member, gdp, population

dim_product (商品维度)
- hs_code, hs_name_cn, hs_name_en, hs_chapter, hs_section

dim_time (时间维度)
- date_key, year, quarter, month, week, day, is_holiday

dim_tariff (关税维度)
- hs_code, partner_country, base_rate, rcep_rate, fta_rate
- rule_of_origin, valid_from, valid_to
```

## 七、AI模型方案

### 7.1 模型清单

| 模型 | 用途 | 框架 | 性能要求 |
|------|------|------|---------|
| LSTM | 贸易额时序预测 | PyTorch | MAPE<15% |
| Prophet | 季节性趋势分解 | Prophet | 训练<5min |
| K-Means | 商品聚类 | SKlearn | 训练<1min |
| DBSCAN | 异常点检测 | SKlearn | 实时 |
| Isolation Forest | 贸易风险预警 | SKlearn | 实时 |
| 规则引擎 | RCEP关税计算 | 自研Python | 准确率>98% |
| DeepSeek/Qwen | 对话式分析 | 本地部署 | 响应<5s |

### 7.2 训练数据

- 历史贸易数据(2010-2024)：约500万条
- 商品分类标准：HS编码体系
- 国家分类：东盟10国+中国+全球主要贸易伙伴
- 关税数据：RCEP/东盟自贸区/双边FTA多源数据

### 7.3 模型评估

- 时序预测：MAE, RMSE, MAPE
- 聚类分析：Silhouette Score, Davies-Bouldin Index
- 异常检测：Precision, Recall, F1
- 规则引擎：人工校验准确率
